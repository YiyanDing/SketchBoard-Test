## [在线画板项目][1]简介
>此项目是Mega Project List中的第二个web项目[onlie white board][2]
![onlie white board][101]

## 功能特色
1. 支持多种字体和颜色
2. 支持高分辨率屏幕
3. 邀请其他用户，协同创作
4. 完成后保存PDF或者PNG
## 效果演示
![效果演示][102]
## 使用说明
```
git clone git@github.com:WJ941/onlineWhiteBoard.git
npm install
node server.js
```
